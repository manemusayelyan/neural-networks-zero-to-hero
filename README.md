# Neural Networks: Zero to Hero (Personal Learning Repo)

This repository documents my journey through Neural Networks: Zero to Hero by Andrej Karpathy.

The purpose of this repo is to build a deep, first-principles understanding of neural networks by reimplementing core concepts from scratch, experimenting with architectures, and carefully documenting insights along the way.

## What’s Inside
* Personal notes summarizing key ideas
* From-scratch neural network implementations
* Visualizations to better understand training behavior
